Dashboard Refresh Demo

How to Test Dynamic Values

This demo shows how the dashboard values change with each refresh:

  1. Click the button below to go to the dashboard
  2. Note the current values displayed
  3. Refresh the dashboard page (F5 or Ctrl+R)
  4. Observe how the values have changed
  5. Repeat to see different values each time

How It Works

The system generates realistic document data with:

  • Random document counts (100-150 documents)
  • Random upload dates within the last 30 days
  • Random file sizes (100KB - 10MB)
  • Realistic views and download counts
  • Varied document categories and statuses

Every hour, completely new data is generated to keep the system fresh!
